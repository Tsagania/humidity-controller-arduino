ΓΙΑ ΤΗ ΒΑΣΗ ΔΕΔΟΜΕΝΩΝ
	Ανοίγουμε το XAMPP και ενεργοποιούμε Apache και MySQL
	Πατάμε Admin στο MySQL
	Κάνουμε import το SQL αρχείο


ΓΙΑ ΤΟ ARDUINO SKETCH 
	Απλά ανοίγουμε το περιβάλλον του arduino και κάνουμε Open File
        Αν έχεις κάνει το κύκλωμα με Arduino, πρέπει να ανεβάσεις (Ανέβασμα) τον κώδικα στην πλακέτα


ΓΙΑ ΤΟΝ ΚΩΔΙΚΑ PYTHON
	Αν έχουμε εγκαταστήσει την Python στον υπολογιστή:
		Ανοίγουμε το CMD απο το start menu και κάνουμε copy paste μια μια τις εντολές για να κατεβάσουμε δύο βιβλιοθήκες
	Αν δουλεύουμε από το Anaconda και το spyder: 
		Ανοίγουμε το Anaconda Promp από την έναρξη και κάνουμε copy paste μια μια τις εντολές για να κατεβάσουμε δύο βιβλιοθήκες
	ΕΝΤΟΛΕΣ	
		pip install mysql-connector-python
		pip install pyserial
	Στη γραμμή 19:  ser = serial.Serial('ΟΝΟΜΑ PORT', 9600) πρέπει στη πρώτη παράμετρο να βάλω το ενεργό PORT του arduino. Το σύστημα πρέπει να είναι ενεργό και το port αναγράφεται στο TOOLS ως COM*  όπου *= 0 ή 1 ή 2 ή 3... κοκ.
        Ανοίγεις ένα περιβάλλον της επιλογής σου που τρέχει κώδικας Python, και τρέχεις το πρόγραμμα. Έτσι ανεβαίνουν τα δεδομένα στη βάση δεδομένων.


ΓΙΑ ΤΟΝ ΚΩΔΙΚΑ PHP
        Έχεις ανοιχτή τη βάση δεδομένων(Ανοίγουμε το XAMPP και ενεργοποιούμε Apache και MySQL)
        Φορτώνεις τον φάκελο ARDUINO_PHP στον φάκελο htdocs του xampp. 
        Μπαίνεις σε έναν browser και γράφεις localhost/ARDUINO_PHP/ , πατάς ENTER και voilà!!
